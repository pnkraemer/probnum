Dirac¶

class probnum.probability.Dirac(support, random_state=None)[source]¶

Bases: probnum.probability.Distribution

The Dirac delta distribution.

This distribution models a point mass and can be useful to represent numbers as random variables with Dirac measure. It has the useful property that arithmetic operations between a Dirac random variable and an arbitrary RandomVariable acts in the same way as the arithmetic operation with a constant.

Note, that a Dirac measure does not admit a probability density function but can be viewed as a distribution (generalized function).

Parameters

support (scalar or array-like or LinearOperator) – The support of the dirac delta function.

See also

Distribution

Class representing general probability distributions.

Examples

>>> from probnum.probability import RandomVariable, Dirac
>>> dist1 = Dirac(support=0.)
>>> dist2 = Dirac(support=1.)
>>> rv = RandomVariable(distribution=dist1 + dist2)
>>> rv.sample(size=5)
array([1., 1., 1., 1., 1.])

random_state¶

Random state of the distribution.

This attribute defines the RandomState object to use for drawing realizations from this distribution. If None (or np.random), the global np.random state is used. If integer, it is used to seed the local RandomState instance.
