Distribution¶

class probnum.probability.Distribution(parameters=None, pdf=None, logpdf=None, cdf=None, logcdf=None, sample=None, mean=None, var=None, random_state=None)[source]¶

Bases: object

A class representing probability distributions.

This class is primarily intended to be subclassed to provide distribution-specific implementations of the various methods (logpdf, logcdf, sample, mean, var, …). When creating a subclass implementing a certain distribution, make sure to override generic operations (addition, multiplication, …) to represent the properties of the distribution. The overriden methods should also update the parameters of the distribution. This allows arithmetic operations on instances of Distribution and RandomVariable.

Parameters

  • parameters (dict) – Dictionary of distribution parameters such as mean, variance, et cetera.

  • pdf (function) – Probability density or mass function.

  • logpdf (function) – Log-probability density or mass function.

  • cdf (function) – Cumulative distribution function.

  • logcdf (function) – Log-cumulative distribution function.

  • sample (function) – Function implementing sampling. Must have signature sample(size=()).

  • mean (function) – Function returning the mean of the distribution.

  • var (function) – Function returning the variance of the distribution.

  • random_state (None or int or RandomState instance, optional) – This parameter defines the RandomState object to use for drawing realizations from this distribution. If None (or np.random), the global np.random state is used. If integer, it is used to seed the local RandomState instance. Default is None.
