asrandomvariable¶

probnum.probability.asrandomvariable(obj)[source]¶

Return obj as a RandomVariable.

Converts scalars, (sparse) arrays or distribution classes to a :class:RandomVariable.

Parameters

obj (array-like or LinearOperator or scipy.stats.rv_continuous or scipy.stats.rv_discrete) – Argument to be represented as a :class:RandomVariable.

Returns

rv – The object obj as a as a :class:RandomVariable.

Return type

RandomVariable

See also

RandomVariable()

Class representing random variables.

Examples

>>> from probnum.probability import asrandomvariable
>>> M = np.array([[1,2,3],[4,5,6]], dtype=np.int32)
>>> asrandomvariable(M)
<2x3 RandomVariable with dtype=int32>
