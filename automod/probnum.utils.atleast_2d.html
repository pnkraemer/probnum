atleast_2d¶

probnum.utils.atleast_2d(*rvs)[source]¶

View inputs as arrays with at least two dimensions.

Parameters

arys2, .. (arys1,) – One or more random variables or array-like sequences. Non-array inputs are converted to arrays or random variables. Arrays or random variables that already have two or more dimensions are preserved.

Returns

rvs – An array, random variable or a list of arrays / random variables, each with a.ndim >= 2.

Return type

array-like or list
